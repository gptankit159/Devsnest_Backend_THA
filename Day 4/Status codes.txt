## Status Codes

1xx -> Information response -> request was received.
2xx -> success -> request was accepted and performed.
3xx -> redirection
4xx -> client side erros
5xx -> server side errors

200 -> All ok
201 -> new resource was created.
204 -> No need to return the information.

304 -> Browser caching.

400 -> missing info in request
401 -> authentication failure
403 -> permission denied

404 -> resource not found
410 -> resource no longer available and will not be available.

500 -> server internal error
